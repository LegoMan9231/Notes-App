Notes: Users will be able to write and format their notes in the way they wish. This is the basis of the whole model.

Accounts: Users will be able to create accounts to store their note instances(Desks) and templates.

AI (Dependent on investment): If AI can be used, allowing it to generate different note templates or to take written 
notes and convert it into a note taking style the user wishes. This is not likely to be fully implemented however it 
is large enough to take note for the future.

Note Templates: With user created note templates, people can share their different ways of thinking using the tools
 provided. This would allow less of a learning curve for new users over time.
 
Collaboration: Users will be able to invite others to their projects to collaborate with them.

Sign in with google: Using a users google account to sign into the website
